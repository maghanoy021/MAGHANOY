All about this project



This Django project was developed by Joan Maghanoy to show how
CRUD (Create, Read, Update, Delete) works for a blog system.


The goal of this project is to demonstrate how Django simplifies web
development by combining powerful backend logic with clean, user-friendly design.
It serves as a foundation for learning how modern web applications manage data, user
interaction, and dynamic content efficiently.
